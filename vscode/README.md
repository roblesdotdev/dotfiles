# VSCode for web development

## Extensions

```
# Vim
ext install vscodevim.vim

# Dotenv
ext install mikestead.dotenv

# Eslint
ext install dbaeumer.vscode-eslint

# Prettier
ext install esbenp.prettier-vscode

# Night Owl(theme)
ext install sdras.night-owl

# Icons
ext install file-icons.file-icons

# Tailwind 
ext install bradlc.vscode-tailwindcss

# PostCSS
ext install csstools.postcss

# Path Intellisence
ext install christian-kohler.path-intellisense

# Intellij Key Binding
ext install k--kato.intellij-idea-keybindings

# Live Server
ext install ritwickdey.LiveServer

# Autoremane Tags
ext install formulahendry.auto-rename-tag

# MDX
ext install silvenon.mdx

# TODO highlight
ext install wayou.vscode-todo-highlight
```
